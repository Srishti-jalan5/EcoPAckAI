# EcoPackAI Dashboard Documentation

## Overview
The EcoPackAI Dashboard is a Business Intelligence (BI) tool designed to visualize and analyze data related to sustainable packaging recommendations. It provides users with insights into eco-friendly materials, packaging options, and environmental impact metrics.

## Features
- Interactive data visualizations to track packaging recommendations.
- User-friendly interface for accessing analytics.
- Integration with the backend API to fetch real-time data.
- Responsive design for accessibility on various devices.

## Setup Instructions
1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/ecopackai.git
   cd ecopackai/dashboard
   ```

2. **Install Dependencies**
   Ensure you have Node.js installed, then run:
   ```bash
   npm install
   ```

3. **Run the Dashboard**
   Start the development server:
   ```bash
   npm start
   ```
   The dashboard will be available at `http://localhost:3000`.

## Usage
- Navigate through the dashboard to explore various analytics related to sustainable packaging.
- Use the interactive charts and graphs to gain insights into material usage and recommendations.
